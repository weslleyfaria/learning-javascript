Exercício 003

Tratamento de Dados

Tipos de dados:
        > number
            > Infinity
            > NaN
        > string
        > boolean
        > null
        > undefined
        > object
            > array
        > function

        let n1 = window.prompt('Digite um número: ')        
        var n2 = window.prompt('Digite outro número: ')
        let result = n1 + n2
        window.alert('Soma de '+n1+' + '+n2+' = '+result)

        O resultado apresentado será uma concatenação!

        Para a soma acontecer, é necessário converter os 
        valores pois o comando window.prompt envia para a
        variável uma string.

        number + number : o JS soma
        string + string : o JS concatena

        Conversão de String para Number

        Number.parseInt(n)
        Number.parseFloat(n)
        Number(n) - simplificado

        let n1 = Number.parseInt(window.prompt('Digite um número: '))
        var n2 = Number(window.prompt('Digite outro número: '))
        let result = n1 + n2
        window.alert('Soma de '+ n1 +' + ' + n2 + ' = ' + result)

        Conversão de Number para String

        String(n)
        n.toString()

        Formatando Strings

        Entre aspas simples ou aspas duplas concatenamos 
        utilizando o + entre strings e variáveis.

        Ex:
        window.alert('Soma de '+ n1 +' + ' + n2 + ' = ' + result)

        Utilizando o template string, dentro do comando alert 
        entre crases (``), referenciamos da seguinte forma: ${n}

        Ex:
        window.alert(`A soma de ${n1} + ${n2} é igual a [${result}]`)
